# DnD App - Android Version

Приложение для игроков Dungeons & Dragons 5e, содержащее справочники по заклинаниям, умениям, цитаты табакси и заметки.

## Функции

- **Заклинания D&D 5e** - полный справочник заклинаний с поиском и детальным просмотром
- **Умения (Feats)** - справочник умений персонажей с поиском по названию и категории
- **Цитаты табакси** - забавные цитаты от кошачьих персонажей с кнопкой случайной цитаты
- **Заметки** - создание, редактирование и удаление заметок для кампаний
- **Настройки** - выбор темы приложения и информация о приложении

## Технологии

- Kotlin
- Android SDK
- Material Design 3
- Navigation Component
- RecyclerView
- ViewBinding

## Установка и запуск

### Предварительные требования

1. **Android Studio** - последняя версия
2. **Java 8** или выше
3. **Android SDK** - API level 24+

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd dnd_app_android
```

2. Откройте проект в Android Studio

3. Синхронизируйте Gradle файлы

4. Подключите Android устройство или запустите эмулятор

5. Запустите приложение:
   - Нажмите "Run" в Android Studio, или
   - Выполните команду: `./gradlew installDebug`

### Проблемы с Gradle

Если возникают проблемы с Gradle wrapper:

1. Убедитесь, что у вас установлен Gradle
2. Или скачайте gradle-wrapper.jar и gradlew файлы
3. Или используйте Android Studio для автоматической настройки

## Разработка

### Добавление новых функций

1. Создайте новый фрагмент в `java/com/example/dndapp/`
2. Добавьте layout файл в `res/layout/`
3. Обновите навигацию в `res/navigation/nav_graph.xml`
4. Добавьте пункт меню в `res/menu/activity_main_drawer.xml`

### Стили и темы

Приложение использует Material Design 3. Основные цвета и темы определены в:
- `res/values/colors.xml`
- `res/values/themes.xml`

### ✅ **Реализованные функции**
1. **Реальные данные** - полная база заклинаний, умений и цитат
2. **Поиск** - поиск по названию, школе, классу и описанию заклинаний
3. **Детальные экраны** - подробная информация о заклинаниях
4. **Управление заметками** - создание, редактирование и удаление
5. **Навигация** - полноценная навигация между экранами
6. **Material Design 3** - современный UI с карточками и анимациями

### 🚀 **Возможные улучшения**
1. **Локальное хранение** - SharedPreferences для настроек и заметок
2. **Фильтрация** - расширенные фильтры для заклинаний
3. **Избранное** - сохранение избранных заклинаний
4. **Темы** - полная поддержка светлой/темной темы
5. **Экспорт** - экспорт заметок в файл
6. **Звуки** - звуковые эффекты для цитат
